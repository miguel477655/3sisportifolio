Cálculo de Raiz Quadrada

Este programa calcula a raiz quadrada de um número fornecido pelo usuário.

Dicas sobre o código

  • math.sqrt(): Função do módulo math para calcular raiz quadrada.
  • int(input()): Converte a entrada do usuário em número inteiro.
  • Módulo math: Precisa ser importado para usar funções matemáticas.
  • f-strings: Permite formatação de strings com variáveis.
  • :.2f: Formata o número para mostrar 2 casas decimais.
